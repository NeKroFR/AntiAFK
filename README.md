# Install from source:

1) makes sure you installed [GO](https://go.dev/doc/install)
2) clone the project:
```
git clone https://github.com/NeKroFR/AntiAFK.git
cd AntiAFK/
sh install.sh
```
3) compile the binary:
```
go build AntiAFK.go
```
